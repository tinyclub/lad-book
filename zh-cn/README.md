
# Linux 考古笔记

2017 年 10 月，[泰晓科技](http://tinylab.org) 发起并成立了“Linux 考古队”，志在共同学习和研究历史版本的 Linux 内核，输出学习笔记并汇总成册。

这个小册子就是用来记录“考古队”的各项考古发现并及时分享给社区。
